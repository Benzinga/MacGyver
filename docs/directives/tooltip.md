
Tooltip
===
Tooltip directive  
  
### Parameters
**mac-tooltip**  
Type: `String`  
Text to show in tooltip  
  
**mac-tooltip-direction**  
Type: `String`  
Direction of tooltip  
  
**mac-tooltip-trigger**  
Type: `String`  
How tooltip is triggered  
  
**mac-tooltip-inside**  
Type: `Boolean`  
Should the tooltip be appended inside element
